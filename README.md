# 🤖 AI Chat Mobile App

Aplicação mobile de chat com Inteligência Artificial desenvolvida com React Native.  
O app permite que usuários conversem com uma IA em tempo real, recebendo respostas rápidas e inteligentes.


---

## 🚀 Funcionalidades

- 💬 Chat em tempo real com IA
- ⚡ Respostas rápidas e otimizadas
- 🧠 Integração com API de Inteligência Artificial
- 📱 Interface moderna e responsiva
- 🎯 Suporte a múltiplos tipos de perguntas
- 🔄 Scroll automático das mensagens

---

## 🛠️ Tecnologias

- React Native
- TypeScript
- Expo
- API da OpenAI
